# 🎯 Meu Portfólio Profissional - Landing Page

Este projeto é uma **Landing Page de Apresentação Profissional**, desenvolvida como uma **atividade acadêmica da faculdade**. O objetivo principal é praticar e testar meus conhecimentos em desenvolvimento web, especialmente na criação de **landing pages modernas, responsivas e bem estruturadas**.

Trata-se de um projeto **puramente educacional**, sem fins comerciais, criado para consolidar habilidades técnicas e evolução prática na área de tecnologia.

---

## 📖 Sobre o Projeto Acadêmico

A proposta deste trabalho foi desenvolver uma página do tipo “cartão de visitas digital”, com foco em:

- 👩‍💻 **Quem sou eu**: Apresentação pessoal, acadêmica e profissional  
- 🚀 **Habilidades Técnicas**: Tecnologias e ferramentas em estudo e uso  
- 🎯 **Áreas de Interesse**: Suporte Técnico, QA, Análise de Dados e Desenvolvimento  
- 💼 **Projetos**: Espaço para exibição de exercícios e aplicações práticas  
- 📬 **Contato**: Links para redes profissionais e comunicação  


---

## 🛠️ Tecnologias Utilizadas

Foram utilizadas tecnologias fundamentais do desenvolvimento web:

- **HTML5**: Estruturação semântica da página  
- **CSS3**: Estilização, responsividade e organização visual  
- **JavaScript** *(quando aplicável)*: Interatividade e melhorias na experiência do usuário  
- **Google Fonts**: Personalização tipográfica para identidade visual  

---

## 🚀 Funcionalidades

- 🎨 Prática de **design de landing page**
- 📱 Desenvolvimento de **layout responsivo**
- 🧩 Organização de seções de forma clara e funcional
- 💡 Aplicação de conceitos de UI/UX básicos
- 🔗 Simulação de um **portfólio profissional real**

---

## 🎯 Objetivo Acadêmico

Este projeto tem como finalidade exclusivamente acadêmica, servindo para:

- Testar conhecimentos adquiridos na faculdade  
- Praticar estruturação de páginas web  
- Desenvolver habilidades em HTML, CSS e JavaScript  
- Evoluir na criação de interfaces modernas
